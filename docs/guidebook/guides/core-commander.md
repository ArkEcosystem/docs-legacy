# Core Commander

![Core Commander](https://i.imgur.com/UipxOEF.png)

**A shell script to easily setup, configure and maintain an Ark Core.**

## Installation

```bash
git clone https://github.com/ArkEcosystem/ARKcommander
cd ARKcommander
bash ARKcommander.sh
```
