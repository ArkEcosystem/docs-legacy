# Retrieve an unconfirmed transaction

[[toc]]

## Endpoint

```
GET /api/transactions/unconfirmed/{id}
```

## Path Parameters

| Name | Type   | Description                                        | Required           |
|------|:------:|----------------------------------------------------|:------------------:|
| id   | string | The identifier of the transaction to be retrieved. | :white_check_mark: |

## Response

```json
{}
```
