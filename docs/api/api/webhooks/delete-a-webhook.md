# Delete a webhook

[[toc]]

## Endpoint

```
DELETE /api/webhooks/{id}
```

## Path Parameters

| Name | Type   | Description                                  | Required           |
|------|:------:|----------------------------------------------|:------------------:|
| id   | string | The identifier of the webhook to be deleted. | :white_check_mark: |

## Response

```json
{}
```
